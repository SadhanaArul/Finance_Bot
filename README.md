## 📌 Overview
Finance Bot is a web-based personal finance tracking application built using Streamlit. It helps users manage income, track expenses, and visualize financial data through interactive dashboards.

## ✨ Features
- ➕ Add and manage income & expenses
- 📊 Interactive charts and financial insights
- 📅 Track transaction history
- 📈 Expense breakdown visualization
- 💡 Simple and user-friendly interface

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
